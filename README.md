## 🌟 기술 스택

- 프레임워크: React 19
- 언어: TypeScript
- 빌드 도구: Vite
- 패키지 관리자: pnpm
- 스타일 가이드: ESLint, Prettier
- 스타일링: Tailwind CSS
- 유틸리티: cn (clsx + tailwind-merge 조합으로 구현)
- 컨테이너화: Docker, Docker Compose

## 🌻 설치 및 실행 방법

```bash
docker-compose up -d
```

- 브라우저에서 접속 `localhost:3000` 에서 확인 가능
